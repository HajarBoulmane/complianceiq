# ComplianceIQ

Document intelligence platform for regulatory compliance analysis of SME contracts.
Combines a RAG pipeline (LangChain.js + ChromaDB) with standalone LLM contract analysis.

## Stack
- Backend: Node.js / Express
- Frontend: React
- Vector DB: ChromaDB
- Relational DB: PostgreSQL
- Infra: Docker, Terraform, AWS
